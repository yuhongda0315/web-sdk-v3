## web-sdk-v3

>目录结构

```
├── dist 
├── release 发版记录
├── bugfix bug 记录
├── scripts 打包脚本
├── src
    └── channel
      └── provider.js channel 适配层
      └── desktop
      └── web 
    └── modules
      └── chatroom Chatroom SDK 输出
      └── im IM SDK 
    └── packages SDK 内部使用的外部工具包
    └── utils
      └── request 请求工具类
        └── wx.js 
        └── normal.js 
        └── provider.js
      └── socket 连接工具类
        └── ws.js 
        └── comet.js
        └── provier.js
      └── emitter.js 事件触发器
      └── observer.js 观察者
      └── utils.js 工具类输出
├── test 
```
